

iowsc by wiki908@github

对已解密的早期 AdvHD 引擎脚本 wsc 应用的处理工具
不支持 unicode 文件名, 不支持 2GiB 以上文件 (其他大小视机器内存而定)
不提供提取和封装时的转码功能——可以使用笔者之前编写的 wcpc 工具批量转码

如果要处理新版的 AdvHD 脚本, 可适当修改代码

引用:
wcpc:  https://github.com/wiki908/wcpc (用于文本批量转码)
AdvHD: https://github.com/Inori/FuckGalEngine/tree/master/AdvHD (用于引擎资源提取)
